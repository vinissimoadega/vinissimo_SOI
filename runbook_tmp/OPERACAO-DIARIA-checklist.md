# Checklist Diário de Operação

## Abertura

- Abrir `/dashboard`
- Revisar estoque crítico
- Revisar pendências de base
- Abrir `/crm`
- Revisar fila do dia

## Durante o dia

- Cadastrar cliente novo quando houver identificação suficiente
- Atualizar cliente existente quando houver contexto útil
- Lançar toda venda real
- Lançar toda compra real
- Usar ajuste manual só com motivo claro
- Abrir tarefa quando houver ação futura

## Fechamento

- Conferir se todas as vendas entraram
- Conferir se todos os clientes novos entraram
- Revisar pós-venda pendente
- Revisar avaliação pendente
- Revisar reativação pendente
- Registrar incidentes e exceções
